A command line tool and Go library for converting arbitrary shell output (with ANSI) into beautifully rendered HTML.

# Usage
$ cat input.raw | terminal-to-html > out.html
# Getting Started
$ go get github.com/buildkite/terminal/cmd/terminal-to-html
# or just download a pre-compiled binary:
$ open "https://github.com/buildkite/terminal/releases"
# Grab the code, read the docs, contribute!
$ open "https://github.com/buildkite/terminal"
